## ❌ Se der erro:

1. Verifique se tem **Node.js** instalado:
   ```bash
   node --version
   npm --version
   ```

2. Se precisar reinstalar:
   ```bash
   npm install --legacy-peer-deps
   npx expo start --web
   ```

3. Se continuar com erro, veja `README.md` para mais detalhes.
